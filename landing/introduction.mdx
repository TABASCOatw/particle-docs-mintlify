## What is Particle Network?

Particle Network radically simplifies user experience through Wallet Abstraction and Chain Abstraction. Through its Wallet Abstraction SDKs, developers can enable one-click onboarding into smart accounts through social logins.

## Key Products

Particle Network current product suite features numerous SDKs made to simplify authentication and onboarding:

<CardGroup cols={2}>
  <Card title="Particle Connect" icon="link" href="../api-reference/connect">
    Unified modal aggregating both social logins (through Particle Auth) and wallet connection.
  </Card>
  <Card title="Particle Auth" icon="shield" href="../api-reference/auth">
     Onboard users with social authentication using popular platforms like Google, X, and so on.
  </Card>
  <Card title="Account Abstraction" icon="credit-card" href="../api-reference/aa">
    Full-stack account abstraction SDK driving the implementation of smart accounts, including features like gasless transactions and session keys.
  </Card>
  <Card title="BTC Connect" icon="btc" href="../api-reference/btc">
     Connect Bitcoin-native wallets, such as UniSat or Xverse, to applications built on EVM-compatible chains.
  </Card>
</CardGroup>
