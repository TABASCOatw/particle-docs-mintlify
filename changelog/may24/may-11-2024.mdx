## BTC Connect
- Added 2.1.0 implementation which supports [EIP-1271](https://eips.ethereum.org/EIPS/eip-1271).
- Introduced new methods: `personal_sign` and `eth_signTypedData_v4`.
- Accounts using lower version implementations will be upgraded automatically.
